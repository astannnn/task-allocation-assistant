# Task Allocation Assistant

A Software Engineering project for team and project management.

The system is designed as an intelligent project management assistant that supports task allocation, workload analysis, delayed task reassignment, and conflict detection using multi-criteria heuristic logic.

## Main Features

- Project management
- Team member profiles
- Skill management
- Task management
- Task required skills
- Automatic task allocation
- Delayed task reassignment
- Workload analysis
- Assignment conflict detection

## Technology Stack

- Python
- FastAPI
- SQLAlchemy
- SQLite
